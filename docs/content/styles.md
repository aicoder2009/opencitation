# Citation Styles

OpenCitation supports four styles. Use the style selector on the citation page to switch at any time — the formatted output updates instantly.

## Choosing a style

If your instructor or publisher specified a style, use that. If not, a rough guide: APA for psychology and social sciences; MLA for literature and humanities; Chicago for history; Harvard for sciences and UK/Australian institutions.

Citations saved to a List are stored with all their raw field data — you can change the style at any time and the citation will be re-formatted correctly.

## APA 7th Edition

| | |
|---|---|
| **Organization** | American Psychological Association |
| **Common in** | Psychology, education, social sciences |
| **In-text** | `(Author, Year)` or `Author (Year)` |

**Book example**

```
Author, A. A. (Year). Title of work: Capital letter also for subtitle. Publisher.
```

**Journal article example**

```
Author, A. A., & Author, B. B. (Year). Title of article. Journal Name, volume(issue), page–page. https://doi.org/xxxxx
```

## MLA 9th Edition

| | |
|---|---|
| **Organization** | Modern Language Association |
| **Common in** | Literature, humanities, arts |
| **In-text** | `(Author Page)` — e.g., `(Smith 42)` |

**Book example**

```
Author Last, First. Title of Work. Publisher, Year.
```

**Journal article example**

```
Author Last, First. "Title of Article." Journal Name, vol. X, no. X, Year, pp. X–X.
```

## Chicago 17th Edition

| | |
|---|---|
| **Organization** | University of Chicago Press |
| **Common in** | History, fine arts, humanities |
| **In-text** | Footnote / endnote superscript, or `(Author Year)` in author-date variant |

**Book example**

```
Author Last, First. Title of Work. Place: Publisher, Year.
```

**Journal article example**

```
Author Last, First. "Title of Article." Journal Name Volume, no. Issue (Year): Page–Page.
```

## Harvard

| | |
|---|---|
| **Organization** | No single governing body — varies by institution |
| **Common in** | Sciences, social sciences, UK/Australia universities |
| **In-text** | `(Author, Year)` — e.g., `(Smith, 2020)` |

**Book example**

```
Author, A.A. (Year) Title of work. Place of publication: Publisher.
```

**Journal article example**

```
Author, A.A. and Author, B.B. (Year) 'Title of article', Journal Name, Volume(Issue), pp. Page–Page.
```
